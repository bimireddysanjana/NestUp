# NestUp

This project converts raw CSV data into a 3D model of a wall, a cabinet, and its internal parts. 
Each row in the dataset represents an object with its dimensions and position. 
I processed the data using Python in Google Colab, cleaned the values, and used the Level field to distinguish between the wall, cabinet, and planks. 
Then, I generated a 3D visualization using matplotlib, where each object is represented as a box placed in its correct position.
The final output clearly shows the complete structure with proper hierarchy and placement.
